## Slide 3: The Solution

# twodirect = Search → Find → Go

### วิธีใช้ง่ายๆ 3 ขั้นตอน:

```
1️⃣ SEARCH    พิมพ์ชื่อ หรือ ถ่ายรูปสินค้า
      ↓
2️⃣ FIND      เห็นว่าสาขาไหนใกล้บ้านมีของ
      ↓
3️⃣ GO        นำทางตรงไป หยิบของได้เลย
```

### ผลลัพธ์:
✅ ลูกค้าได้ของ
✅ ร้านได้ยอดขาย
✅ ไม่เสียเวลาเดินทางเปล่า
